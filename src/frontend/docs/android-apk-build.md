# Building Android APK for NARC EYE

This guide explains how to package the NARC EYE PWA as an Android APK using Trusted Web Activity (TWA) technology.

## What is TWA?

Trusted Web Activity (TWA) is a way to package Progressive Web Apps (PWAs) as native Android applications. The app runs your web content in a full-screen Chrome Custom Tab without any browser UI, providing a native app experience.

Before building the APK, ensure you have the following installed:
